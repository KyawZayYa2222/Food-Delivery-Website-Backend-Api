## Installation
- clone the repo
- git clone https://github.com/KyawZayYa2222/Food-Delivery-Website-Backend-Api.git
- composer install or composer update
- cp .env.example .env
- setup .env file
- php artisan key:generate
- php artisan storage:link
- php artisan migrate
- php artisan serve
